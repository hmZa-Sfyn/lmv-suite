## Options

### domain
- **Type:** string
- **Description:** Domain to resolve
- **Required:** Yes

### record_type
- **Type:** string
- **Description:** DNS record type (A, AAAA, MX, NS, TXT, CNAME)
- **Required:** No
- **Default:** A

### nameserver
- **Type:** string
- **Description:** Custom nameserver to use
- **Required:** No
