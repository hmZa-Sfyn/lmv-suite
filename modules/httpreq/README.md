# httpreq

## Description
Makes HTTP requests to a target

## Metadata
- **Type:** bash
- **Author:** LanManVan Team
- **Version:** 1.0.0

## Tags
networking, web, http

## Options

### host
- **Type:** string
- **Description:** Target host (without http://)
- **Required:** Yes

### method
- **Type:** string
- **Description:** HTTP method (GET, POST, etc)
- **Required:** No
- **Default:** GET

### path
- **Type:** string
- **Description:** URL path
- **Required:** No
- **Default:** /
