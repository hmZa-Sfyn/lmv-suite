# bluetooth-scanner

## Description
Scan for nearby Bluetooth devices, display device names, signal strength, and device types

## Options

### duration
- **Type:** integer
- **Description:** Scan duration in seconds
- **Required:** No
- **Default:** 10

### device_type
- **Type:** string
- **Description:** Filter by device type (all, phone, laptop, headset, etc.)
- **Required:** No
- **Default:** all

### show_rssi
- **Type:** boolean
- **Description:** Show signal strength (RSSI)
- **Required:** No
- **Default:** True
